# brute_rack

HTTP API for the [Brute](https://github.com/general-intelligence-systems/brute) coding agent. Rack app with JSON and SSE streaming endpoints. Deploy with [Falcon](https://github.com/socketry/falcon).

## Usage

Please see the [project documentation](https://general-intelligence-systems.github.io/brute_rack/) for more details.

  - [Getting Started](https://general-intelligence-systems.github.io/brute_rack/guides/getting-started/index) - This guide walks you through setting up brute_rack to expose the Brute agent over HTTP.
